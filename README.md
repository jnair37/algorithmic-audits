# Algorithmic Audits Experiments Repository

This contains the combined experiment notebooks from the fall semester. These are consolidated into the latest iteration of each method, and any previous iterations are included in the previous-iterations folder. 

All of these notebooks should run as Colab or Jupyter notebooks.

-combine_interface.ipynb contains the LLM + VL interface for resume screening and image captioning
-ngrok_test.ipynb contains the original vision interface
-proto_vit.ipynb contains prototypical parts network experiments
-daam6.ipynb contains diffusion experiments
